# 评审团、小法庭AI评审工具

## 项目简介

本工具由美团的“小美评审团”有感而发，使用LLM Agent来评审差评、退款等是否合理，并和大众评审结果进行对比。

## 版权声明

本项目仅用于大模型训练、Agent开发的交流，不指向任何一个平台，因此不提供爬虫等代码。

部分数据采用了[美团](https://www.meituan.com/)的“小美评审团”在抖音、B站等平台的分享数据。

## 项目结构说明

- `backend/`：后端服务，服务端获取题目和评审结果、上传新的题目
- `frontend/`：前端界面，用户可以在这里查看题目、提交评审结果
- `docs/`：项目文档，一些设计细节、示例题目
- `agents/`: LLM Agent代码，提取用户意图和商户意图，以及客观的事实信息，推演评审结论（适合展示/不适合展示/适合退款/不适合退款）
- `finetune/`: 微调代码，包含数据处理和模型训练脚本

## 项目部署说明

使用Docker Compose. 可以直接在服务器部署一份服务。
